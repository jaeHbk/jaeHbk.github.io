Agricultural Productivity Forecast

Frontend (Google Maps API)

  • The user interacts with a frontend that enables selection of a bounding box region.

Backend Service

  • Tiling Module (tiling.py): Reads tiling data (from a text file) and computes intersections with the selected geographic area.
  • Product Query Module (productquery.py): Queries the Copernicus Data Catalogue (via the OData API) for products matching the area, date (including day filtering), and cloud cover criteria.
  • Download Module (S3 Download): Uses the temporary S3 credentials retrieved via the S3 Keys Manager Service to download satellite imagery from the EO Data S3 storage using boto3.
  • ML Model Analysis Module: After downloading, the imagery can be processed by ML models for analysis.

External Services

  • Copernicus Data Catalogue (OData API): Provides product metadata based on geographic and temporal queries.
  • Authentication Server (CDSE Identity): Provides access tokens required for querying product metadata and obtaining S3 credentials.
  • S3 Keys Manager Service: Issues temporary S3 credentials.
  • EO Data Storage (S3): Stores the satellite imagery that is downloaded.

Local Storage

  • Downloaded imagery is saved into a local folder structure, organized by year, month, and tile, which can then be passed to downstream processes (e.g., ML analysis).

Built with: Python, AWS, EOData API
